# TumanIO
# Данный проект предназначен для вендингового аппарата "Генератор сухого тумана для автомобиля", выполнен в среде Arduino.ide на языке C++

TumanIO Pro — это интеллектуальное устройство для генерации сухого тумана, предназначенное для профессиональной обработки салонов автомобилей. Устройство автоматизирует процесс: клиент выбирает аромат, производит оплату через POS-терминал, после чего система нагревает жидкость до нужной температуры и распыляет её в течение 60 секунд. Эффект сохраняется до 14 дней.

✨ Особенности
3 независимых канала с разными ароматами

Управление через Telegram — мониторинг, статистика, настройки

POS-терминал для безналичной оплаты (подготовлена интеграция)

LED-матрица 64×32 для отображения информации и рекламы

Автоматический подогрев для работы при отрицательных температурах

Защита от перегрева (программная + механический термостат)

Обновление по воздуху (OTA) через ArduinoOTA

Сохранение статистики в энергонезависимой памяти

Автономный режим при потере связи

💻 Программная часть
Архитектура прошивки
Прошивка построена на машине состояний, что обеспечивает чёткую логику работы и устойчивость к сбоям. Основные состояния:

ОЖИДАНИЕ — приветствие, опрос кнопок, проверка уровня жидкости

ВЫБОР АРОМАТА — фиксация выбора, запрос оплаты

ОПЛАТА — ожидание подтверждения от POS-терминала (эмуляция)

НАГРЕВ — включение ТЭНа, контроль температуры до 85°C

ГОТОВНОСТЬ — ожидание нажатия кнопки СТАРТ

РАБОТА — включение насоса на 60 секунд

ЗАВЕРШЕНИЕ — очистка, запись статистики

АВАРИЯ — обработка критических ошибок (перегрев, низкий уровень)

Используемые библиотеки
WiFi.h, WiFiClient.h — подключение к сети

UniversalTelegramBot.h — управление Telegram-ботом

OneWire.h, DallasTemperature.h — работа с DS18B20

Preferences.h — энергонезависимое хранение настроек

ArduinoOTA.h — обновление по воздуху

Telegram-бот
Проект включает двух ботов:

Пользовательский (для владельца) — управление, статистика, настройки.

Сервисный (для разработчика) — массовое обновление, диагностика.

Команды пользовательского бота:

/menu — главное меню

/status — текущее состояние системы

/analytics — статистика продаж (день/неделя/месяц)

/control — управление (запуск без оплаты, сброс статистики, перезагрузка)

/config — настройка цены, имён ароматов, таймингов, Wi-Fi

/info — информация об устройстве (IP, MAC, версия)

Автоматические уведомления:

Низкий уровень жидкости

Перегрев

Ежедневный/еженедельный отчёт
